Welcome to Git-Pulse

A local analytics dashboard tailored to low-spec hardware. Analyze directory logs, calculate change velocities, map active hour matrices, and visualize distribution graphs.

How to get started:

  • 1. Input the absolute path of a Git repository in the **"Track New Repository"** panel (top left).
  • 2. Click **"+ Register Repo"** to instantly crawl logs and render your dashboard.
  • 3. Add multiple repositories in the same panel to switch between dashboards on-demand.
